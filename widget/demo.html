🤖 Alice for Arkansas

Chat Widget Demo

Welcome to the Alice Chat Widget

Alice is an AI assistant that provides accurate ALICE (Asset Limited, Income Constrained, Employed) data for Arkansas counties. This chat widget can be embedded on any website.

👉 Try it out! Click the chat button in the bottom-right corner and ask Alice questions like:
  • "What's the ALICE rate in Pulaski County?"
  • "Tell me about Washington County demographics"
  • "What's the statewide ALICE rate for Arkansas?"
  • "Compare Benton and Sebastian counties"

📋 How to Embed

Add this single line of code to your website before the closing </body> tag:

<script src="http://localhost:3000/widget/alice-chat-widget.js" data-api-url="http://localhost:3000/api/chat" data-title="Chat with Alice" data-subtitle="Ask about Arkansas ALICE data" data-color="#4F46E5" data-position="bottom-right"></script>

⚙️ Configuration Options

Customize the widget appearance using data attributes:

  • data-api-url - Your Alice API endpoint
  • data-title - Widget header title
  • data-subtitle - Widget header subtitle
  • data-color - Primary color (hex code)
  • data-position - Position: bottom-right, bottom-left, top-right, top-left
